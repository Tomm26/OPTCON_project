# Dettagli Implementazione Progetto OPTCON

## Analisi dei File e Confronto con Richieste

### File dal Codice del Professore
I file forniti dal professore sono esempi di implementazione di:
1. **Gradient Method - Scalar & Vectorial** (1_gradient_method_scalar.py, 2_gradient_method_vectorial.py)
2. **Newton's Method** (3a_newton_method_unconstrained.py, 3b_newton_method_armijo.py, 3c_newton_method_compare.py)
3. **QP constrained** (4b_QP_constrained.py)
4. **SQP** (5_SQP.py, 5b_SQP_equilibria.py)
5. **LQR e MPC** (main_dlqr_tracking.py, main_MPC.py, ecc.)

Questi file sono stati usati come riferimento per l'implementazione delle tecniche di ottimizzazione e controllo.
